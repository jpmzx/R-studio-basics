# R-studio-basics

## Dependencies
- R version 4.0.2
- R Studio version 1.3.1093

## R packages
- ggplot2
- swirl
- lattice
- car
- flmport
- tseries

## Installing packages
Use install.packages() from R studio console

```R
> install.packages(c("ggplot2","swirl","lattice","car","flmport","tseries"))
```
